# 错题笔记应用 — 功能需求设计

## 概述

一款支持手写的小米 6 Pro 平板错题笔记应用，用于学习考试类错题的记录、整理和复习巩固。纯本地使用，备份至百度网盘或夸克网盘。

---

## 一、核心功能

### 1.1 错题录入

- **题目来源**：拍照或上传图片，调用 AI（DeepSeek/Minimax）识别提取文字
- **题目展示**：题目图片在上方，下方留出手写答题区域
- **标注功能**：可在题目图片上手写画线、标注重点
- **答案解析**：录入正确答案、解析文字
- **答题类型**：选择题（设置正确选项）/ 大题（需 AI 对比）
- **分类管理**：选择科目（文件夹）+ 知识点标签（可多标签）
- **复习计划**：导入时自动生成 7/14/21 天复习计划（从导入日算起）

### 1.2 复习交互

- **复习列表**：进入"今日复习"，显示当天所有该复习的错题（按导入时间从先到后排列），每条标注科目/知识点标签
- **复习流程**：
  - 选择题：显示题目图片 + 单选按钮 → 用户选答案 → 提交 → 自动判断对错
  - 大题：显示题目图片 + 手写答题区 → 用户手写作答 → 提交 → AI 对比分析（指出符合/不符合正确答案的点）→ 用户手动标记对/错
- **跳过功能**：可手动跳过某一轮复习
- **记录对错**：选择题自动记录，大题用户手动标记

### 1.3 错题管理

- **组织方式**：
  - 左侧：科目文件夹树 → 展开后显示知识点
  - 右侧：错题列表（按时间排序，支持搜索）
- **标签系统**：每条错题可有多个知识点标签

### 1.4 统计分析

- **错误次数统计**：列表展示错 1 次 / 错 2 次 / 错 3 次的题
- **知识点正确率**：柱状图按科目/知识点展示正确率（同一题的 3 次复习算 3 题）

---

## 二、数据结构

### 错题条目

| 字段 | 说明 |
|------|------|
| id | 唯一标识 |
| subject | 科目 |
| tags | 知识点标签列表 |
| questionImage | 题目图片路径 |
| correctAnswer | 正确答案（选择题存选项，大题存文字）|
| explanation | 解析 |
| questionType | 选择题 / 大题 |
| createdAt | 导入时间 |

### 复习记录

| 字段 | 说明 |
|------|------|
| wrongCount | 错误次数 |
| reviews | 各轮复习记录（含对错、时间、是否跳过）|

---

## 三非功能性需求

- 纯本地存储
- 手动备份至百度网盘或夸克网盘
- 支持小米 6 Pro 手写笔
- 无商业用途
